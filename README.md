# Retail-Financial-Analysis
This dashboard provides a detailed analysis of retail financial performance, integrating sales, profit, and expense data into a single interactive view. It helps stakeholders track KPIs, identify trends, and make informed business decisions. KPI Cards: ➤ Total Revenue ➤ Net Profit ➤ Expense Ratio Visuals: ➤ Monthly Sales &amp; Profit Trends (Line Chrt)
